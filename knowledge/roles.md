# Roles

Six roles make up every palette. Define them by job, not by hue.

- **background** — the furthest-back surface; the page. Light mode: near-white but rarely pure `#ffffff`. Dark mode: near-black but rarely pure `#000000`.
- **surface** — raised panels, cards, inputs. A small, deliberate step in lightness off the background, not a different hue.
- **text** — primary reading color on background and surface. Must clear the contrast targets in both modes.
- **muted** — secondary text, captions, placeholders. Lower contrast than text, but never below the AA floor for body copy.
- **accent** — the one color that carries brand and action (links, primary buttons, focus). Must be unmistakably distinct from the neutrals. If it reads as "just another gray," it has failed.
- **border** — hairlines and dividers. Low contrast by design, but still perceivable against the surface it sits on.

Light and dark are both first-class. A palette is not "a light theme plus an inverted copy" — each mode is composed to feel intentional on its own.
